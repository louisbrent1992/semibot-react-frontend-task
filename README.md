# SemiBot Frontend Development Task

## Description

React web page that displays album api data.

## Getting Started

### Dependencies

- styled-components
- react-icons
- axios
- @mui/material
- @mui/icons-material
- @emotion/react
- @emotion/styled

### Installing

1. Download or Fork project
2. `npm install` to install dependencies

### Executing program

1. `npm run start` to run project
